=== КАК ТЕСТИРОВАТЬ САЙТ НА КОМПЬЮТЕРЕ ===

Шаг 1: Открой Terminal
  (Finder → Программы → Утилиты → Terminal)

Шаг 2: Введи команду и нажми Enter:
  cd ~/Desktop/atomy-site && python3 -m http.server 8080

Шаг 3: Открой браузер и перейди по адресу:
  http://localhost:8080

Всё! Сайт работает как на хостинге.
Terminal можно свернуть — он работает в фоне.
Когда закончишь тестирование — закрой Terminal.

=== КАК ДОБАВИТЬ НОВЫЙ ТОВАР ===

1. Скопируй файл: data/products/ШАБЛОН_НОВОГО_ТОВАРА.json
2. Переименуй его в: data/products/КАТЕГОРИЯ/новый_id.json
   (например: data/products/krasota/kr200.json)
3. Заполни все поля (инструкции внутри файла)
4. Добавь в data/index.json новую строку с основными данными товара
5. Создай папку photos/КАТЕГОРИЯ/новый_id/
6. Положи туда фото: 1.jpg, 2.jpg ... (до 8 штук)
7. Обнови поле "photoCount" в JSON файле товара
8. Загрузи обновлённые файлы на хостинг

=== СТРУКТУРА ПАПОК ===

atomy-site/
  index.html          ← главный файл сайта (не трогай)
  data/
    index.json        ← список всех товаров (обновляй при добавлении)
    products/
      zdorovye/       ← категория Здоровье
        zd15.json     ← файл товара
        ...
      krasota/        ← категория Красота
      gigiena/        ← Личная гигиена

      pitanie/        ← Продукты питания
      aza/            ← AZA Молл
  photos/
    zdorovye/
      zd15/           ← папка фото товара
        1.jpg         ← фото 1
        2.jpg         ← фото 2
        ...
    krasota/
    ...
